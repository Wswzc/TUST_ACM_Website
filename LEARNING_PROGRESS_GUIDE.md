# 学习进度查看指南

## 如何查看你的学习进度

生成学习计划后，你可以通过以下方式查看你的学习进度：

### 方法一：通过导航链接

1. 在首页的"制定学习计划"区域，点击 **"📊 查看我的进度"** 按钮
2. 页面会自动滚动到"我的学习进度"仪表板

### 方法二：生成计划后直接跳转

1. 填写学习计划表单并点击"生成学习计划"
2. 在生成的计划结果中，点击 **"点击这里查看我的学习进度 →"** 链接
3. 页面会自动跳转到进度仪表板

### 方法三：直接访问进度仪表板

在首页URL后添加 `#progress-dashboard-section`，例如：
```
http://localhost:8080/index.html#progress-dashboard-section
```

## 进度仪表板功能

### 📊 学习进度卡片

每个学习路径都有一个进度卡片，显示：

- **学习路径名称**：算法竞赛、C/C++开发、Python开发等
- **学习计划信息**（如果已制定）：
  - ⏰ 每周学习时间
  - 🎯 预计完成月数
  - 📊 学习阶段数量
- **进度条**：显示当前学习进度百分比
- **操作按钮**：
  - 如果已有计划：显示"开始学习"按钮
  - 如果没有计划：显示"制定学习计划"按钮，点击后自动跳转到计划器并预选该路径

### 📈 统计面板

显示你的整体学习统计：

- 🎯 **已获成就**：完成学习里程碑获得的成就数量
- 📚 **学习路径**：正在学习的路径数量
- ⏱️ **学习天数**：从开始学习到现在的总天数

## 数据存储

所有学习进度和计划数据都保存在浏览器的 LocalStorage 中，包括：

- 学习计划详情（每周时间、阶段、里程碑等）
- 学习进度（完成的主题、最后活动时间）
- 获得的成就

**注意**：清除浏览器缓存会删除这些数据，请谨慎操作。

## 功能特性

### 1. 自动保存
- 生成学习计划后自动保存到本地
- 标记完成主题后自动更新进度
- 无需手动保存或登录账号

### 2. 实时更新
- 生成新计划后，进度仪表板会自动刷新
- 进度卡片会显示最新的计划信息

### 3. 智能导航
- 点击"制定学习计划"按钮会自动跳转到计划器
- 如果该路径没有计划，会自动预选该路径

### 4. 成就系统
- 完成学习里程碑会解锁成就
- 成就会以通知形式弹出显示
- 所有成就都会记录在统计面板中

## 常见问题

### Q: 为什么看不到我的学习进度？

A: 请确保：
1. 已经生成了至少一个学习计划
2. 页面已完全加载（等待几秒）
3. 浏览器支持 LocalStorage（现代浏览器都支持）
4. 没有清除过浏览器缓存

### Q: 如何重置学习进度？

A: 打开浏览器开发者工具（F12），在 Console 中执行：
```javascript
localStorage.removeItem('tust-acm-study-plans');
localStorage.removeItem('tust-acm-learning-progress');
location.reload();
```

### Q: 可以导出学习进度吗？

A: 目前学习进度保存在本地，可以通过浏览器开发者工具查看和导出 LocalStorage 数据。未来版本会添加导出功能。

### Q: 学习进度会同步到其他设备吗？

A: 目前不支持跨设备同步，数据仅保存在当前浏览器中。如需在多设备使用，建议使用同一浏览器并开启同步功能（如 Chrome 同步）。
