### 4. 学习计划器 📋

**实现文件：** `src/assets/js/study-planner.js`

**核心功能：**
- ✅ 个性化计划生成
- ✅ 智能时间调整算法
- ✅ 4 阶段学习规划
- ✅ 每周任务清单
- ✅ 学习里程碑设置
- ✅ 完成时间预测

**学习阶段模板：**

每个学习路径包含 4 个阶段：
1. **基础阶段** - 掌握基本概念
2. **进阶阶段** - 深入核心技术
3. **提高阶段** - 项目实战
4. **冲刺阶段** - 综合应用

每个阶段包含：
- 预计时长
- 学习主题列表
- 每周任务清单
- 实践项目建议

**智能算法：**
```javascript
// 根据每周学习时间调整计划周期
if (weeklyHours < 10) {
    adjustedMonths *= 1.5;  // 延长 50%
} else if (weeklyHours > 20) {
    adjustedMonths *= 0.8;  // 缩短 20%
}
```

**用户价值：**
- 科学的学习规划
- 明确的学习目标
- 合理的时间安排
- 可执行的任务清单
