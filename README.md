# TUST-ACM 算法实验室官网

天津科技大学 TUST-ACM 算法实验室官方网站，提供系统的算法学习路径、丰富的学习资源和智能的学习追踪功能。

## ✨ 主要特性

### 🎯 学习路径系统
- **6 条专业学习路径**：算法竞赛、C/C++开发、Python开发、前端开发、后端开发、AI/机器学习
- **详细课程大纲**：每条路径包含 4 个学习阶段，循序渐进
- **职业发展指导**：明确的职业方向和就业建议
- **实战项目示例**：从入门到进阶的项目案例

### 📊 学习进度追踪
- **可视化进度**：实时显示每条路径的完成百分比
- **学习统计**：追踪学习天数、已获成就、活跃路径
- **本地存储**：无需注册，数据保存在浏览器本地
- **进度导出**：支持导出学习报告

### 🏆 成就系统
- **5 级成就**：从"迈出第一步"到"路径大师"
- **即时反馈**：解锁成就时弹窗通知
- **持久化保存**：成就永久记录
- **激励学习**：通过游戏化方式提升学习动力

### 📚 学习资源中心
- **精选资源**：18 本经典书籍、15 个学习平台、18 个开发工具、8 门在线课程
- **分类清晰**：按学习路径和资源类型组织
- **详细介绍**：每个资源都有描述和推荐理由
- **外部链接**：直接访问资源网站

### 📋 学习计划器
- **个性化计划**：根据时间安排生成定制计划
- **智能调整**：基于每周学习时间优化学习周期
- **阶段规划**：4 个学习阶段，每阶段有明确任务
- **里程碑设置**：关键节点提醒和激励

## 🚀 快速开始

### 本地运行

1. **克隆项目**
```bash
git clone https://github.com/your-repo/tust-acm-website.git
cd tust-acm-website
```

2. **安装依赖**
```bash
npm install
```

3. **启动开发服务器**
```bash
gulp
```

4. **访问网站**
打开浏览器访问 `http://localhost:3000`

## 📖 使用指南

### 学习者指南

1. **选择学习路径** - 浏览 6 条学习路径，选择适合的方向
2. **制定学习计划** - 填写时间安排，生成个性化计划
3. **开始学习** - 点击"开始学习"，系统自动追踪进度
4. **追踪进度** - 勾选完成的主题，解锁学习成就
5. **查找资源** - 访问资源中心，获取优质学习材料
