# TUST-ACM 算法实验室官网

天津科技大学 TUST-ACM 算法实验室官方网站，一个功能完整的学习管理系统，提供系统的算法学习路径、丰富的学习资源、智能的学习追踪和数据分析功能。

## ✨ 核心功能

### 📚 学习路径系统
- **6 条专业学习路径**：算法竞赛、C/C++开发、Python开发、前端开发、后端开发、AI/机器学习
- **详细课程大纲**：每条路径包含 4 个学习阶段，循序渐进
- **职业发展指导**：明确的职业方向和就业建议
- **实战项目示例**：从入门到进阶的项目案例

### 📊 学习进度追踪
- **可视化进度**：实时显示每条路径的完成百分比
- **学习统计**：追踪学习天数、已获成就、活跃路径
- **本地存储**：无需注册，数据保存在浏览器本地
- **进度导出**：支持导出学习报告

### 🏆 成就系统
- **5 级成就**：从"迈出第一步"到"路径大师"
- **即时反馈**：解锁成就时弹窗通知
- **持久化保存**：成就永久记录
- **激励学习**：通过游戏化方式提升学习动力

### 📖 学习资源中心
- **精选资源**：59 个优质资源（书籍、平台、工具、课程）
- **分类清晰**：按学习路径和资源类型组织
- **详细介绍**：每个资源都有描述和推荐理由
- **外部链接**：直接访问资源网站

### 📅 学习计划器
- **4 步向导**：路径选择 → 水平评估 → 目标设定 → 时间安排
- **个性化计划**：根据时间和水平生成定制计划
- **智能调整**：基于每周学习时间优化学习周期
- **阶段规划**：4 个学习阶段，每阶段有明确任务

### ✅ 学习打卡系统
- **每日打卡**：记录每天的学习情况
- **连续统计**：显示连续打卡天数，激励持续学习
- **打卡日历**：可视化展示打卡记录
- **成就徽章**：根据打卡天数获得不同等级的徽章

### 📈 数据可视化分析
- **学习路径完成度**：饼图展示各路径进度
- **学习时长趋势**：折线图显示最近7天学习时间
- **每周活动统计**：柱状图展示一周内每天的学习时长
- **知识点掌握**：雷达图多维度评估知识掌握情况
- **真实数据**：所有图表数据来自 LocalStorage，无假数据

### 🌙 深色模式
- **一键切换**：点击浮动按钮即可切换主题
- **自动跟随系统**：首次访问时跟随系统主题设置
- **偏好保存**：记住用户的主题选择
- **全站适配**：所有页面统一支持深色模式

### 🏅 竞赛日程
- **8 大平台**：Codeforces、AtCoder、LeetCode、洛谷等
- **实时倒计时**：显示距离比赛开始的时间
- **智能筛选**：按平台和时间筛选比赛
- **日历导出**：支持导出 ICS 格式日历文件

### 🎯 求职导向
- **校招时间线**：展示关键时间节点
- **面试准备**：系统化的面试准备建议
- **简历指导**：提供优秀简历模板
- **常见面试题**：整理高频面试题目

## 📖 使用指南

### 学习者完整流程

1. **选择学习路径** - 浏览 6 条学习路径，选择适合的方向
2. **制定学习计划** - 使用 4 步向导生成个性化学习计划
3. **开始学习** - 点击"开始学习"，系统自动追踪进度
4. **每日打卡** - 访问我的仪表板，记录每天的学习
5. **追踪进度** - 勾选完成的主题，解锁学习成就
6. **查看分析** - 通过数据可视化图表了解学习情况
7. **查找资源** - 访问资源中心，获取优质学习材料
8. **参加竞赛** - 查看竞赛日程，参加算法比赛

### 主要页面

- **首页** (`index.html`) - 学习路径展示和计划制定
- **我的仪表板** (`my-dashboard.html`) - 打卡、数据分析、进度追踪
- **资源中心** (`resources-hub.html`) - 学习资源推荐
- **竞赛日程** (`contests.html`) - 算法竞赛时间表
- **求职导向** (`career-path.html`) - 校招和面试指导
- **关于我们** (`about.html`) - 实验室介绍
- **联系我们** (`contact.html`) - 联系方式

## 💾 数据存储说明

所有用户数据都存储在浏览器的 LocalStorage 中，包括：

- **学习进度** (`tust-acm-learning-progress`) - 每条路径的完成情况
- **学习计划** (`tust-acm-study-plans`) - 用户制定的学习计划
- **打卡记录** (`tust-acm-checkin-records`) - 每日打卡数据
- **主题偏好** (`tust-acm-dark-mode`) - 深色/浅色模式选择

### 数据真实性保证

所有数据可视化图表都使用真实数据，无假数据：
- 学习时长数据来自打卡记录和学习计划
- 每周活动数据来自打卡统计
- 知识点掌握数据来自学习进度的加权计算

详见 `数据说明.md` 和 `数据已修复.md`。

## 🐛 常见问题

### Q: 数据会丢失吗？
A: 数据保存在浏览器的 LocalStorage 中，除非清除浏览器缓存，否则不会丢失。

### Q: 可以导出数据吗？
A: 学习计划支持导出为 JSON 格式，其他数据可通过浏览器开发者工具导出。

### Q: 深色模式会影响性能吗？
A: 不会，深色模式只是 CSS 样式的切换，对性能影响极小。

### Q: 图表数据从哪里来？
A: 所有图表数据来自 LocalStorage 中的学习记录，会自动计算和更新，无假数据。

### Q: 如何重置所有数据？
A: 打开浏览器控制台（F12），执行：
```javascript
localStorage.clear();
location.reload();
```

### Q: 竞赛数据如何更新？
A: 目前使用模拟数据，可通过修改 `contest-schedule.js` 集成真实 API。
